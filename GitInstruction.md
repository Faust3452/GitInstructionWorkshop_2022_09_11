## Создание и базовая работа с локальным репозиторием.

### Что такое репозиторий и инструкция по созданию локальных репозиториев.
Репозиторий - система хранения файлов с отслеживанием версионности

Для создания репозитория необходимо перейти в локальную папку, планируемую для использования в качестве репозитория, и выполнить команду ***`git status`*** в терминале. Эта команда позволит узнать, является ли данная папка репозиторием, вновь созданная папка заведомо репозиторием не будет. Поэтому необхожимо набрать в терминале ***`git init`*** для инициализации локального репозитория.

После выполнения этой команды локальный репозиторий будет инциализирован.

Также есть возможность клонировать существующий репозиторий. Сделать это можно следующей командой:
```
git clone <адрес репозитория>
```
Репозиторий создан!!!
